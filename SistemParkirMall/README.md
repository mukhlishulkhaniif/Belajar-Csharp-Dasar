
# PROJECT 02: SISTEM PARKIR MALL

### 📝 Deskripsi
Program sederhana berbasis konsol untuk menghitung biaya parkir berdasarkan jenis kendaraan dan durasi parkirnya.

---

### 💻Fitur Utama
* Menerima input jenis kendaraan.
* Menerima input durasi parkir kendaraan.
* Menghitung biaya yang harus dibayar secara presisi.

---

### 💡Logika Penilaian
* **Mobil**: Rp5.000 / jam
* **Motor**: Rp2.000 / jam
* **Rumus**: Total = Tarif x Durasi

---

### 🛠️Teknik C# yang Digunakan
* **Selection Logic**: Menggunakan switch-case atau if-else untuk menentukan tarif berdasarkan pilihan kendaraan.
* **Arithmetic**: Perkalian sederhana untuk mendapatkan total bayar.
* **Data Validation**: Memastikan input pilihan kendaraan sesuai dengan menu yang tersedia.
* **While Loop**: Digunakan untuk menjaga program tetap berjalan selama user belum memilih untuk keluar.
