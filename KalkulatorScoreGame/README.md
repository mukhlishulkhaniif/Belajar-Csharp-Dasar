# PROJECT 01: KALKULATOR SCORE GAME

### 📝 Deskripsi
Program sederhana berbasis konsol untuk menghitung rata-rata skor pemain dari tiga babak dan memberikan penilaian otomatis.

---

### 💻Fitur Utama
* Menerima input nama pemain dan skor tiap babak.
* Menghitung nilai rata-rata secara presisi.
* Menentukan grade hasil akhir secara otomatis.

---

### 💡Logika Penilaian
* **Skor > 80**: Hebat
* **Skor 50 - 80**: Boleh Juga
* **Skor < 50**: Belajar Lagi

---

### 🛠️Teknik C# yang Digunakan
* **Input & Output**: Console.ReadLine & Console.WriteLine
* **Variabel**: String, Int, Double
* **Logika**: If-Else Statement
* **Struktur**: Static Method dengan Parameter
