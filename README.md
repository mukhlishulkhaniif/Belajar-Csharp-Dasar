# 📘 Belajar C# Dasar

Repository ini berisi kumpulan project sederhana dan latihan logika yang saya kerjakan selama proses mempelajari bahasa pemrograman C#. 

Tujuan dari repo ini adalah untuk mendokumentasikan pemahaman saya mengenai konsep dasar pemrograman hingga implementasi logika sederhana.

---

### 📂 Daftar Project

1. **Kalkulator Score Game**
   * Menghitung rata-rata skor dari 3 babak permainan.
   * Sistem penilaian otomatis: **Hebat** (>80), **Boleh Juga** (50-80), dan **Belajar Lagi** (<50).
   * Menggunakan logika percabangan untuk menentukan hasil akhir.

2. **Sistem Manajemen Parkir**
   * Menghitung tarif berdasarkan jenis kendaraan: **Mobil** (Rp5.000/jam) dan **Motor** (Rp2.000/jam).
   * Fitur kalkulasi otomatis: `Tarif x Durasi Jam`.
   * Implementasi `while loop` agar user bisa menghitung biaya berulang kali tanpa merestart program.
  
3. **Sistem Kasir Sederhana**
   * Menghitung total belanja untuk item Buku, Pensil, dan Pulpen.
   * Fitur Diskon Otomatis: Potongan 10% jika total belanja di atas Rp50.000 menggunakan logical operator.
   * Teknik Modular: Menggunakan methods dengan arguments untuk memisahkan logika perhitungan matematika.
   * Validasi Input: Memastikan transaksi hanya tercatat jika pilihan barang valid (harga > 0).
     
---

### 🛠️ Materi yang Sudah Dipelajari

* **Dasar & Input**: Variabel, User Input, Logical Operator.
* **Aritmatika**: Operator Aritmatika, Assignment, Increment, Decrement.

### 🚀 Cara Menjalankan
1. Clone repository ini.
2. Buka folder project yang diinginkan.
3. Jalankan file `.csproj` menggunakan Visual Studio atau gunakan perintah `dotnet run` melalui terminal.
